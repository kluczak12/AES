# Kryptoanaliza
## Funkcje
* **Szyfrowanie i Deszyfrowanie:** Pełna implementacja cyklu AES.
* **Operacje na blokach:** Obsługa kluczowych etapów algorytmu:
    * `SubBytes` (S-Box)
    * `ShiftRows`
    * `MixColumns`
    * `AddRoundKey`
* **Zarządzanie kluczami:** Generowanie harmonogramu kluczy (Key Expansion).
